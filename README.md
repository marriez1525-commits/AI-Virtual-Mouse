# 🖱️ AI Virtual Mouse



## 💡 How It Works

1. Webcam captures live video.
2. MediaPipe detects hand landmarks.
3. Finger positions are extracted.
4. Gestures are recognized.
5. Corresponding mouse actions are executed using PyAutoGUI.
